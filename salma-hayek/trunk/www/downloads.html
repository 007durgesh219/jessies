Linux (x86)

For Ubuntu and other Debian-based Linux distributions, there's .deb">.deb which will add to your GNOME applications menu. (For command-line or non-GNOME users, there's /usr/bin/.)

For Fedora and other RPM-based Linux distributions, there's .rpm">.rpm, which is equivalent to the Debian package.

Mac OS

For Mac OS 10.4, there's .dmg">.dmg which contains a universal binary application that you can simply drag and drop to install.

MS Windows

For Windows, there's a Windows installer .msi"> which requires that you've already installed Cygwin, including Ruby, yourself but is otherwise complete and leaves you with an icon on your desktop.

We have more information about setting up Cygwin.

(Why did we choose Cygwin? Mainly for Terminator's benefit. NT's built-in POSIX subsystem won't let you launch native Windows applications or interact with the network, so that wasn't an option. Cygwin, on the other hand, even contains the Unix98 pseudo-terminal support which we use to support window resizing, not to mention containing a port of one of the most widely used ssh programs.)
